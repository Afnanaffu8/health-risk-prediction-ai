# 📌 Project Overview
This project is an AI-powered healthcare analytics system that predicts a patient's health risk based on key parameters like Age, BMI, Glucose Level, and Blood Pressure.

It also includes a Generative AI chatbot that provides intelligent health-related responses using RAG (Retrieval-Augmented Generation).


#🎯 Features

- 🔍 Health Risk Prediction using Machine Learning
- 📊 Risk Percentage Calculation (Low / High Risk)
- 📈 Graph Visualization of Health Parameters
- 💬 GenAI Chatbot for Healthcare Queries
- 🇮🇳 Insights related to Indian health trends
- 🎨 Clean and modern Streamlit UI



# 🛠️ Tech Stack

- Python
- Streamlit
- Machine Learning (Custom Model)
- Matplotlib (Data Visualization)
- RAG (Retrieval-Augmented Generation)
- Generative AI (LLM-based responses)


#🧪 How It Works

1. User enters:
   - Age
   - BMI
   - Glucose Level
   - Blood Pressure

2. Model predicts:
   - Risk Percentage
   - Risk Level (Low / High)

3. Displays:
   - Health insights
   - Graphical representation

4. Chatbot:
   - Answers health-related questions using AI + knowledge base
